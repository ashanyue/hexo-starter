好人没好报的事情一直在重演。<br/>
<br/>
感动中国2005年度人物丛飞，想来大家都不陌生。<br/>
<br/>
1994年至2005年期间，丛飞作为一名普通歌手，义演300多场，义工服务超过6000小时。<br/>
<br/>
他用自己的收入先后资助了贵州、湖南、四川等地的贫困儿童183名，捐助失学儿童和残疾人超过150人，认养孤儿37人，捐助金额超过300万元......<br/>

<br/>

<br/>
丛飞的好心，没换来知恩图报，也没换来好报，只换来一群白眼狼。<br/>

每当看到这样的事件时，既寒心又痛心，寒心被帮助之人的忘恩负义，痛心好人得不到好报。<br/>
<br/>
“好人没好报”的背后，是变坏的世道，是变冷漠的社会。<br/>
<br/>
3<br/>
<br/>
从小到大，在看过大大小小的“好人没好报”事件后，我依然相信这世上还是好人多，也依然坚信：每个人都不是一座孤岛，人心能换人心。<br/>
<br/>
我想起了“水鬼”郭文标的故事。<br/>
<br/>
郭文标是“海上平安民间救助站”的义务救援人员，义务捞尸救人36年来，他睡觉从不关机，几乎不脱衣服、不盖被子，为的就是在有人需要他救援时，他能一秒也不耽误。<br/>
<br/>
他本是一名渔民，因水性极好又熟知海图，他经常抛下手上的活去救人，救的人多了，周边海域只要一出事，就会求救于郭文标。<br/>
<br/>
<br/>
<br/>
郭文标索性放弃了打渔，在政府的帮助下成立了“海上平安民间救助站”，拿出自己的积蓄造救援船，给底下16名队员发放工资。<br/>
<br/>
36年间，他救助遇险人员1240人，不管是14级的台风天气还是打捞离岸100多公里的尸体，他从来都是拿命救人之后，分文不收。<br/>
<br/>
他的见义勇为，每年要赔上几万元的人工费、油费，但换来的是妻子的不理解，以及被他人视作义务。<br/>
<br/>
在尝试做渔家乐补贴救助站被村里人抢了资格后，他也曾泄气过，不明白“为什么救了那么多人，别人就不能为他想一回”甚至决定不再义务救人了，但要他见死不救，始终过不去心里那个坎。<br/>
<br/>
<br/>
<br/>
好在故事最后是皆大欢喜，他成功开张了渔家乐，越来越多的渔民也义务加入到救援队，一有空就来帮忙；<br/>
<br/>
还有人，把郭文标当楷模，也自发组织救援队，参与到了义务救援的行列......<br/>
<br/>
哈维尔曾说：<br/>
<br/>
“我们坚持一件事情，并不是因为这样做了会有效果，而是坚信，这样做是对的。”<br/>
<br/>
做好人行好事，不是为了有好报，而是坚信这样做是对的，为了让自己问心无愧。<br/>
<br/>
丛飞是，江歌是，郭文标亦如是。<br/>
<br/>
善行的力量，就如同一场润物细无声的春雨，在潜移默化之间侵染着人心，滋润着良知，让这个世界一点点地变好。<br/>
<br/>
4<br/>
<br/>
有人会问：<br/>
<br/>
“做好人，就该像丛飞负债资助他人，像郭文标那样不顾性命去义务救人，活成无私奉献的活雷锋吗？”<br/>
<br/>
当然不是。<br/>
<br/>
丛飞、郭文标值得我们尊敬与敬佩，却不意味着做好人就一定得活成他们，但至少，我们遇到需要帮助之人，不要事不关己高高挂起，而是给他们提供力所能及的帮助。<br/>
<br/>
因为，世道变坏，从“好人没好报”开始；世界变冷漠，从人人都不愿意做好事开始。<br/>
<br/>
总有一天，你的孩子也需要帮助，你也会变老。<br/>
<br/>
不要到，这些千千万万需要帮助的弱势群体变成你或者你亲人时，才发现，身边竟无一人肯伸出援助之手。<br/>
<br/>
但行好事，莫问前程。<br/>

<br/>
律师说，事情还有另一种转机是：如果判定他是见义勇为，他就无需承担法律责任了。<br/>
<br/>
我希望，正义能得到伸张，冤屈能得到昭雪，而他，不会成为下一个“好人没好报”之人。<br/>
<br/>
当我们为赵宇呐喊和伸展正义时，不单单是为了他，其实也是为了我们自己，为了我们在需要帮助的时候，得到好人的帮助。<br/>

<br/>
